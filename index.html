Start from the basics of programming with clear, step-by-step lessons. You’ll learn about variables, data types, conditions, loops and functions using beginner-friendly languages like Python and Java. Each lesson includes short examples and small exercises so you can practise as you go. By the end of this section, you’ll feel comfortable reading and writing simple programs on your own.

Build modern applications by combining front-end and back-end skills. Explore HTML, CSS and JavaScript for the browser, then move into server-side development, APIs and basic deployment workflows. You’ll see how real web apps are structured, how data flows between client and server, and how to organize your code so it’s easy to maintain and extend.

Understand how devices talk to each other over local and global networks. This section covers IP addressing, ports, routing, switching and the main network layers. Through diagrams and examples, you’ll learn what actually happens when you open a website, send a message or connect to Wi-Fi. Practical activities and tools like ping and traceroute help you see these concepts in action.

Learn how data is stored, queried and managed in relational databases. You’ll work with tables, primary keys and relationships, and practise writing SQL queries using SELECT, JOIN, GROUP BY and more. Examples are based on realistic datasets, so you can connect what you learn to real applications, reports and projects you might build as an Informatics student.

Build a solid foundation in cybersecurity by exploring common threats, vulnerabilities and defense techniques. Topics include passwords and authentication, encryption, network attacks and safe online practices. You’ll see how security concepts link back to programming and networking, and why secure design matters for every kind of system you work on.

Turn theory into practice with guided projects, quizzes and ready-made study tools. Use templates, checklists and example code to prepare for exams or to build portfolio-ready mini-projects. This section helps you connect all the topics together—programming, networks, databases and security—so you can see the bigger picture and gain confidence in your overall Informatics skills.
